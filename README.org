* 目的

- 簡略モデルをRで実装
- モデルのパラメタが感染拡大にどのように影響を与えるかを実験
- 簡単な相転移(パーコレーションの一種)を確認

* R Script
  - libepidemic.r :: 実験用の自作関数
  - exp1.r :: 感染の推移
  - exp2.r :: 確率的なばらつきの評価
  - exp3.r :: グラフ構造の特性の影響
  - exp4.r :: 感染モデルの特性の影響
  - exp5.r :: 感染拡大の相転移
  - exp6.r :: 相転移の近似的な説明用
  - makepdf.r :: 文書用のpdf作成

* 課題
  まだ誤植や誤解がいろいろありそう．
